### ✅ Share posts with text and images 📝📷
### ✅ Explore trending and top-rated anime 📺🌟
### ✅ Engage in discussions with fellow anime fans 🗣️
### ✅ Personalize their profiles 🎨

# 🚀 Features

### 🔹 Authentication – Secure login and signup 🔑
### 🔹 Post Creation – Share anime-related posts with text & images 📝
### 🔹 Real-time Updates – Seamless interaction using Firestore 🔄
### 🔹 Image Upload – Upload images from the gallery 📷
### 🔹 Profile Customization – Set a username & profile image ✨

# 🛠️ Tech Stack

### 🧑‍💻 React Native – Cross-platform mobile development
### 🔥 Firebase – Firestore (DB), Storage, Authentication
### 🧭 React Navigation – Smooth app navigation

# 🛠️ Installation Guide

 <div>
    <img src="https://media.tenor.com/bU8W2-lHZhYAAAAi/luffy.gif" alt="One Piece Luffy" style="width: 50px; height: auto;">
  </div>
Follow these steps to set up Strawhat locally:

## 1️⃣ Clone the repository ⬇️

```git clone https://github.com/RachitaPant/Strawhat.git```
## 2️⃣ Navigate to the project directory 📂
```cd anime-community-app```

## 3️⃣ Install dependencies 📦

```npm install```
## 4️⃣ Set up Firebase 🔥

-> Create a Firebase project
-> Add configuration details to firebaseConfig.js
-> Enable Firestore and Storage services
## 5️⃣ Run the app ▶️
 For Android:

```npx react-native run-android```
